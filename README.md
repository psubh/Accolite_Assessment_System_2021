# Accolite_Assessment_System_2021

Its an Assessment system to login with google account to create, modify (project , questions) and add quiz to give exam  and  see report card

## Features
1. Sign in via Google SignIn
2. Home page
3. Projects
4. Questions
5. Quiz
6. Exams
7. Result
